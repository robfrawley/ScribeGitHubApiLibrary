Constants

AUTH_URL_TOKEN

Constant for authentication method.

Indicates the default, but deprecated login with username and token in URL.
AUTH_URL_CLIENT_ID

Constant for authentication method.

Not indicates the new login, but allows usage of unauthenticated rate limited requests for given clientid + clientsecret
AUTH_HTTP_PASSWORD

Constant for authentication method.

Indicates the new favored login method with username and password via HTTP Authentication.
AUTH_HTTP_TOKEN

Constant for authentication method.

Indicates the new login method with with username and token via HTTP Authentication.
